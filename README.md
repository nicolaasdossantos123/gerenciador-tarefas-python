# 📋 Gerenciador de Tarefas em Python

Projeto simples de linha de comando desenvolvido para praticar Python na prática.  
O sistema permite gerenciar tarefas, salvando os dados em um arquivo JSON para manter as informações mesmo após encerrar o programa.

---

# Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Contar tarefas pendentes
- Salvar tarefas automaticamente em arquivo JSON
- Carregar tarefas ao iniciar o programa
- Encerrar o programa

---

# Conceitos utilizados

- Lógica de programação  
- Estruturas de repetição (`for`, `while`)  
- Condições (`if / elif / else`)  
- Funções (`def`)  
- Manipulação de listas e dicionários  
- Tratamento de erros (`try / except`)  
- Manipulação de arquivos  
- Uso de JSON (`json.dump` e `json.load`)
